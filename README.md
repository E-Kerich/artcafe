
# Art Café Website

This is a simple, clean, and visually appealing website for an **Art Café**, built using **HTML**, **Bootstrap 5**, and **Google Fonts**.

The website includes sections such as:

* Home (Hero Banner)
* About
* Menu
* Gallery
* Events
* Contact Form

It serves as a starter template for showcasing an artistic coffee shop, blending creativity and functionality.

Feel free to customize and expand it as needed!
